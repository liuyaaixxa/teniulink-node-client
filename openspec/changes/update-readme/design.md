## Context

Teniulink Node 是从 Cherry Studio fork 的 Electron 跨平台 AI 助手应用。项目已完成多项功能开发（PRD1~PRD10），包括：
- 品牌重命名：Cherry Studio → Teniulink Node
- 新增 Teniu云 配置和连接功能
- 新增本地智能网关（原 API 服务器）
- 本地系统配置显示
- HuggingFace 推理服务集成

README.md 需要更新以反映这些变化，同时保留原作者声明。

## Goals / Non-Goals

**Goals:**
- 更新项目名称、描述、品牌标识
- 概述核心功能特性
- 提供简洁的快速上手指南
- 保留原作者声明和感谢

**Non-Goals:**
- 不创建详细的功能文档（链接到单独文档）
- 不修改任何代码逻辑
- 不保留 Cherry Studio 企业版相关内容（本项目是独立 fork）

## Decisions

### 1. README 结构设计

采用简洁的单页结构：
```
1. 品牌标识（Logo + 名称）
2. 简介（1-2 句话）
3. 核心特性（列表形式）
4. 快速开始
5. 技术栈
6. 开发指南（简要）
7. 致谢
8. 许可证
```

**理由**: 保持简洁，详细信息链接到文档站点。

### 2. 功能特性呈现

重点突出已实现的新功能：
- Teniu云 集成
- 本地智能网关
- 多 LLM 提供商支持
- 本地模型支持

### 3. 原作者致谢保留

在致谢部分保留对 Cherry Studio 原作者的感谢声明。

## Risks / Trade-offs

| 风险 | 缓解措施 |
|------|----------|
| 链接失效 | 使用相对路径或稳定的文档链接 |
| 内容过时 | 保持简洁，详细内容放文档站点 |
